<h1> Histórias de Usuário</h1>


```sh
Como usuário 

Quero poder acessar uma página contendo informações úteis

Para poder me informar
```

```sh

Como usuário 

Quero poder ter informações através de gráficos

Para poder ter uma melhor compreenção das estatísticas de minhas consultas

```

```sh
Como usuário 

Quero filtrar as informações por data e/ou por local, código ou amparo

Para fins de objetividade.

```

```sh
Como usuário 

Quero ter acesso a comentários feitos por outros usuários

Para obter informações acerca das infrações
```

```sh
Como usuário 

Quero fazer comentários sobre determinada ocorrência

Para difundir informações acerca das infrações
```

```sh
Como usuário 

Quero poder obter acesso a um mapa de determinada ocorrência

Para melhor vizualizar possíveis locais de infrações
```

```sh
Como usuário 

Quero poder entrar em contato com a equipe de desenvolvimento sempre que necessário

Para possíveis sugestões e/ou reclamações
```
